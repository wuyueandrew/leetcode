1121学习总结
旋转数组这道题，可以通过三种方法：暴力，环状替换，反转。最开始只想到了暴力，学习了题解以后学到一个反转数组的小技巧：双指针，
双指针在数组中是高频出现的技巧（自己写是通过对半反转的，需要多考虑很多），最后，环状替换要注意和元素交换不一样，会多使用一个临时空间。
